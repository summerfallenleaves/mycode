## 测试中修复的 Bug

| Commit | 问题 | 修复 |
|--------|------|------|
| `24d58ae` | `createReactAgent` 收到 Record 而非数组 `.filter()` 崩溃 | Record→`tool()` 数组转换 |
| `25d9aa2` | assistant 消息不保存导致多轮对话上下文丢失 | `fullText` 追加 + `HumanMessage`→`AIMessage` |
| `8cf4b93` | `formatToolResult` 中 `(undefined).trim()` 崩溃 | JSON.parse + 防御性默认 `{}` |
| `1fccf1c` | `output.content` 为 `undefined` 崩溃 | null/undefined 防护 |
| `929e602` | LangChain args 嵌套 `{input: escapedJson}` | 两处解包 |
| `e2716ad` | `\r` 污染 s.input 导致命令不响应 | 过滤 `\r`/`\n` |
| `9913ef9` | `/forget` modal 被 Enter 双事件清除 | `s.forgetList` 专用标志 + 控制字符过滤 |
| `3800168` | `/models` 被 Enter 双事件瞬间关闭 | 500ms Enter 冷却 |
| `87ff988` | `!echo` 面板被 Enter 双事件清除 | 仅 Esc 关闭 |
| `12c2baf` | `!` 命令结果不保存到消息记录 | 对话内联显示 + `addSystemMessage` |

## 架构变更

| 变更 | 描述 |
|------|------|
| LLM 调用 | Vercel AI SDK → LangChain `createReactAgent` + `streamEvents` v2 |
| 工具系统 | `toToolSet()` → `toLangChainTools()` + `tool()` 包装 |
| 会话持久化 | 扁平 `messages[]` → `SessionFile v2`（`turns[].entries[]`，5 种消息类型） |
| Checkpointing | 新增 `MemorySaver`，`thread_id` 会话隔离 |
| 依赖 | 移除 `ai`/`@ai-sdk/*` → 新增 `langchain`/`@langchain/*` |
| 版本 | 0.0.2 → 0.1.0 |
